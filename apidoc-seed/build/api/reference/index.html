  • Description
  • Request
  • Response

Retrieve an organization (id, name, list of user, contact person). A user work to only one organization.

URI Parameters

  • id string

HTTP status code 200

Body

Type: application/json

Example: 
{
    "contact": {
        "id": 1,
        "login": "doe",
        "name": "John Doe"
    },
    "id": 1,
    "name": "AMT",
    "sensors": [
        {
            "description": "Sensors in room 123",
            "id": 2,
            "name": "sensors123",
            "open": true,
            "type": "Temperature"
        }
    ],
    "users": [
        {
            "id": 1,
            "login": "doe",
            "name": "John Doe"
        }
    ]
}

HTTP status code 404

The id was not found.
